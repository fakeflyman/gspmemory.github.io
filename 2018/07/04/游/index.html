游

发表于 2018-07-04 | 分类于 一首诗的时间 | |

两周时间，
一路无言，
我和Ta，
在安静的世界走走停停。
吹温和的风，
淋滂沱的雨，
在诗意小城撞钟，
听记忆久远回响。
旅行，
总多一份浪漫主义的感慨，
于这世界，
更能温柔以待。
